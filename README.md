# Real Estate Management System (REMS)

The **Real Estate Management System (REMS)** is an in-progress software engineering project designed to streamline and automate real estate operations for large-scale commercial property developers.  
This system supports leasing agents, administrative staff, and tenants by providing tools for inventory management, appointment scheduling, rental agreements, invoicing, and maintenance requests.

## üöÄ Core Features (Planned)

### **1. Inventory Management**
- Add, update, and manage property units  
- Track by size, location, rental rate, and classification  
- Search and filter available store spaces  

### **2. Appointment Scheduling**
- Tenant scheduling for property viewings  
- Leasing agent availability management  
- Automatic double-booking prevention  

### **3. Rental Agreements**
- Digital rental applications  
- Electronic lease agreement signing  
- Automatic lease renewals  

### **4. Invoicing & Payments**
- Automated rent invoicing based on payment cycles  
- Utility billing (electricity, water, waste)  
- Overdue notifications and payment tracking  

### **5. Maintenance Management**
- Submit maintenance requests (routine and emergency)  
- Automatic prioritization of critical issues  
- At-fault charges applied to tenant accounts
